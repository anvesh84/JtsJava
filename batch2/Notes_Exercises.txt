There are 2 types of memories as 

	1) STACK

	2) HEAP 


value types will be stored in STACK memory, and reference types will be stored in HEAP memory. 

Boxing and Unboxing : 
______________________

The process of converting value type to reference type known as boxing, and reference type to 
value type known as unboxing. Object is type that which can be used for boxing and unboxing. 

int x;

x="Hello";
x=12;
x=12.5;

Object x;
x="Hello";
x=12;
x=12.5;

Method Overloading : The concept of using the same method name w.r.t. Different data types or different parameters known as method overloading. Here, the method name must be same, but type of parameters or no.of parameters we are passing are different.
